# 📊 Blinkit Sales Analysis Dashboard

An interactive business intelligence dashboard built using Power BI and SQL. The project focuses on analyzing Blinkit sales data to uncover customer behavior, revenue trends, product performance, and business insights through clear and dynamic visualizations.

## ✨ Technologies

`* Power BI`
`* SQL`
`* Excel / CSV Dataset`
`* Data Visualization`
`* DAX`

## 🚀 Features

* Interactive sales dashboard with dynamic filters
* Revenue, profit, and order trend analysis
* Product and category performance insights
* Customer purchasing behavior analysis
* KPI cards for a quick business overview
* Data cleaning and transformation using Power Query

## 📍 The Process

I wanted to build a project that felt close to a real business scenario rather than just creating charts. The goal was to understand how data can help businesses make better decisions. I started by cleaning and transforming the raw dataset, then built relationships between tables and created measures using DAX. After that, I designed an interactive dashboard focused on readability and actionable insights. This project helped me improve my skills in SQL, Power BI, data visualization, and analytical thinking.

## 🎯 Dashboard Insights

* Top performing product categories
* Sales trends over time
* Revenue contribution by outlet type
* Customer order patterns
* KPIs like Total Sales, Average Sales, and Number of Orders

## 🚦 Running the Project

1. Download the `Power BI` (.pbix) file
2. Download the Dataset (.xlsx) file
3. Open it in `Power BI Desktop`
4. Refresh the dataset if needed
5. Explore the interactive dashboard
